# Kimberley Traffic Simulation 🚦

Traffic flow simulation and congestion analysis using AnyLogic.

## Project Overview
This project simulates traffic patterns in Kimberley, South Africa, and analyses congestion points.  
It demonstrates how simulation modelling and data analysis can help improve city traffic flow.

## Features
- Traffic flow simulation  
- Congestion analysis  
- Optimization strategies  

## Tools & Technologies
- AnyLogic (Simulation Modelling)  
- Python / R (Data Analysis)  
- Data Visualization
