## Project Overview
This project simulates traffic patterns in **Kimberley, South Africa**, and analyses congestion points.  
It demonstrates how **simulation modelling and data analysis** can help improve city traffic flow.

---

## Features
- Traffic flow simulation  
- Congestion analysis  
- Optimization strategies  
- Interactive visualization of traffic patterns

---

## Tools & Technologies
- **Simulation:** AnyLogic  
- **Data Analysis:** Python, R  
- **Visualization:** Tableau, Matplotlib, Seaborn  
- **Database / Storage:** SQL (optional if used)
